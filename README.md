# AmorPy

AmorPy adalah aplikasi untuk melakukan kalkulasi amortisasi dengan metode straight-line.

## Fitur

*   GUI
*   Output disimpan dalam format xlsx

## Penggunaan

Download aplikasi pada release lalu jalankan seperti biasa.

# AmorPy

AmorPy is a desktop application for calculating loan amortization schedules using the straight-line method.

## Features

*   **User-Friendly Interface:** A simple Graphical User Interface (GUI) for easy data input.
*   **Amortization Calculation:** Calculates the amortization schedule based on the principal, interest rate, and loan term.
*   **Export to Excel:** Saves the generated amortization schedule in an `.xlsx` file for your records.

## Prerequisites

Before you begin, ensure you have met the following requirements:
*   You have installed Python 3.x.
*   You have a Windows/Linux/Mac machine.

## Installation and Usage

1.  Clone the repository:
    ```sh
    git clone https://github.com/bintanngg/AmorPy.git
    ```
2.  Run the application:
    ```sh
    python main.py
    ```

Alternatively, you can download the executable from the Releases page and run it directly.
